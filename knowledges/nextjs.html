링크

기본적으로

pages 디렉토리 내부의 hierarchy로 url 변경을 통한 페이지 접근이 가능하다. 하지만

network tab을 살펴보면 redirect를 하면서 모든 resource를 다시 받는 것을 알 수 있다.

즉 SSR이 원하는 대로 구현되고 있지 않다.

nextJs는

import Link from 'next/link';

를 제공한다.

이를 활용하면 redirect때도 서버에 재요청을 하지 않는다.
